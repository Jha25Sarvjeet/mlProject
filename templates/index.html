Problem Statement

This project seeks to understand how the student's performance (test scores) is affected by various factors such as:

  • Gender
  • Ethnicity
  • Parental Level of Education
  • Lunch Type
  • Test Preparation Course

By analyzing these factors, we aim to predict the student's performance and uncover patterns in academic success.

About the Project

This machine learning project uses student data to create a model that predicts future test scores based on personal information such as their background, educational environment, and preparation. The goal is to understand the importance of various socio-demographic and academic factors in student achievement.
